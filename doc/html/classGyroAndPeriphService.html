As user connect he/she should see a custom service with a few characteristics: gyro AX, AY, AZ and peripheral control. Select the peripheral control and send the command (0x01, 0x02, 0x03) to turn led on. Detailed Description

Gyroscope service demonstrating GattServer features of mbed OS 6.

The service has 3 gyro-characteristics that contain the G-s values from MPU6050, gain via interrupt operation (MPU is pre-configured for this). A nRF Connect client can subscribe to updates of this characteristics and get notified when one of the value is changed. Clients can also change the value of the peripheral controled characteristic: set 0x01 to turn HIGH level of the appropriate output or 0x00 to set LOW pin level. The UUID of all characteristics was generated using python3 UUID module.
